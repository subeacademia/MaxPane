[B][SIZE=4]MaxPane — Nested docker layouts for REAPER[/SIZE][/B]

[IMG]https://raw.githubusercontent.com/b451c/MaxPane/main/docs/images/maxpane-hero.png[/IMG]

MaxPane is a native C++ extension that lets you build custom multi-pane workspaces inside REAPER. Capture any REAPER window — Media Explorer, FX Browser, Mixer, Actions, or even ReaImGui scripts — into a single tiling container with resizable split panes and tabbed panels.

No scripts, no ReaImGui, no js_ReaScriptAPI required. Pure C++ using REAPER SDK + WDL/SWELL.

[B][SIZE=3]Features[/SIZE][/B]

[LIST]
[*] [B]Flexible split layouts[/B] — split panes horizontally or vertically, up to 16 panes. Drag splitter bars to resize.
[*] [B]Tabbed windows[/B] — multiple windows per pane with tab bar. Drag tabs between panes to rearrange.
[*] [B]14 known REAPER windows[/B] — one-click capture for Media Explorer, FX Browser, Actions, Mixer, Region Manager, Routing Matrix, Track Manager, Big Clock, Navigator, Undo History, Project Bay, Video, Virtual MIDI Keyboard, Performance Meter.
[*] [B]Arbitrary window capture[/B] — grab any open REAPER window, including ReaImGui scripts (ReaMD, etc.).
[*] [B]Workspaces[/B] — save and restore complete layout snapshots with one click.
[*] [B]Favorites[/B] — pin frequently used windows for quick access.
[*] [B]Tab colors[/B] — color-code tabs with 8 palette colors.
[*] [B]Layout presets[/B] — 5 built-in: Two Columns, Left + Right Split, Three Columns, 2×2 Grid, Top + Bottom Split.
[*] [B]Per-project state[/B] — layout saved inside each .RPP file.
[*] [B]Persistent state[/B] — everything survives REAPER restarts.
[*] [B]Dockable[/B] — the container docks into REAPER's native docker system.
[/LIST]

[B][SIZE=3]Installation[/SIZE][/B]

[B]ReaPack (recommended):[/B]
[LIST=1]
[*] Extensions → ReaPack → Import repositories...
[*] Paste:
[CODE]https://raw.githubusercontent.com/b451c/MaxPane/main/index.xml[/CODE]
[*] Extensions → ReaPack → Browse packages → search "MaxPane"
[*] Right-click → Install → Restart REAPER
[/LIST]

[B]Manual install:[/B]
[LIST=1]
[*] Download from [URL=https://github.com/b451c/MaxPane/releases/latest]GitHub Releases[/URL]
[*] Copy to your UserPlugins folder:
[LIST]
[*] macOS: [B]~/Library/Application Support/REAPER/UserPlugins/[/B]
[*] Windows: [B]%APPDATA%\REAPER\UserPlugins\[/B]
[*] Linux: [B]~/.config/REAPER/UserPlugins/[/B]
[/LIST]
[*] Restart REAPER
[*] Actions → "MaxPane: Open Container"
[/LIST]

[B][SIZE=3]Quick start[/SIZE][/B]

[LIST=1]
[*] Run "MaxPane: Open Container" from the Actions menu
[*] Right-click any pane (or click the ▼ button) to open the context menu
[*] Choose a window from the Known Windows list
[*] Split panes via Split Left/Right or Split Top/Bottom
[*] Drag tabs between panes to rearrange
[*] Save your layout via Workspaces → Save Current
[/LIST]
